Important FAQ

What is DOM?

Data representation of the objects that comprise the structure and content of a document on the web.

How do you select an element from DOM?

getElementById, getElementsByClassName, getElementsByTagName, querySelector, etc.

What is event delegation in context of DOM and why is it useful?

A technique where a single event listener is attached to a parent element, rather than multiple listeners being attached to individual child elements.

How do you manipulate an element's attributes and styles using DOM?

By accessing and modifying properties of the element.
